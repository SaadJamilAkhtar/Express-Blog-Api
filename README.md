# Blog API
Backend API of MERN stack Blog App

## DEV ENV
1. Node js
2. Express js

# How to run app:
1. install node js
2. install packages using  
   `npm install`
3. create a .env file in base directory
    1. create env variables  
       Details mentioned Below
4. run file using command  
   `npm start`

## Required ENV variables:
1. NODE_ENV
2. PORT
3. MONGO_URI
4. JWT_SECRET
